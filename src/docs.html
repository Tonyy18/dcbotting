Introduction

dcbotting.com is made by discord users to discord users.

The goal was to make the development of a discord bot as easier as possible. Therefore we converted the programming process to graphical components in order to build a whole discord bot based on them.

dcbotting is not affiliated with discord in any way.

Environment

Another great thing is the size of the environment.

Dcbotting doesn't require you to download or install anything. The entire bot is build by using javascript which runs in any modern web browser.

Once you add your own API Key, it starts hosting the bot in your own computer / browser. Therefore when you close the browser tab, the bot goes down aswell.

But you can always download your current project by clicking the download button in the header.

The data that each event and method includes can be found from the original discord api documentation.

Each component has a button that takes you to the documentation.

Every changes made to the project has to be submitted by pressing the submit button in the header

Bot Token

Bot token can be found from your discord developer portal.

Select one of your applications or create a new one from the top right

From the left sidebar go to the "Bot" section. Build-A-Bot section includes your bot token. The token acts like a password and should be kept as a secret

Invitation link

You will need this link in order to get your bot to join your guild / server.

The link can also be found from your discord developer portal.

Select your application / bot and go to the OAuth2 section from the left sidebar.

Make sure you select the "bot" scope. After that the link will appear.

Permissions

Your bot wont be able to execute some methods without having the corresponding permissions.

The lack of permissions will make your methods to fail and your bot to misfunction. So make sure you select the required permissions.

The editor will provide a "testing" invitation link in the console. This link has a permissions of a adminstrator and should only be used for development.

Components

Everything is a component. You can add new components from the left sidebar.

Every component returns some data. What data each returns can be found from the original discord api documentation

Component includes a documentation button that takes you to the data object

Expression inputs

Components has a expression inputs which can be used to determinate when should the children components be executed.

The values {content} and {[0].content} equals the same because they are accessing the same property but in a different way. Therefore the expression returns True and the methods below will be executed.

You can add as many expressions you want

You can access the data objects returned by components by using curly braces just like in the image. More of this in the Data objects section

Data objects

Data returned by events and methods are stored in an array.

The array is private for each event and stores only the data returned by its children methods.

In the above image you can see the index of a each data object in the event array.

The first [0] index is data returned by the event itself. Rest are objects returned by the executed methods.

The event array in the image would look something like this

            [
                {
                    //Channel create event data
                    id: "749627693783318589",
                    name: "test",
                    ...
                },
                {
                    //data from modify guild method
                },
                {
                    //data from create message method
                }
            ]

Accessing the array / objects

You can get an access to the array by using curly braces {}.

Using the [0] index is mandatory. The inputs in the above image are accessing the same object in a different way.

Programming

The bot is also customizable with javascript

You can add a javascript component as an event or method. Any data returned by the component will be appended into the event array.

The event array in javascript component is stored into data variable.
